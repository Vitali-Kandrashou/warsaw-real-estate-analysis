# Warsaw Real Estate Analysis

## Project Overview

Analysis of residential apartment transactions in Warsaw based on data from the Polish Register of Real Estate Prices and Values (RCN).

## Tech Stack

- Python
- Pandas
- GeoPandas
- PostgreSQL
- SQLAlchemy
- Power BI
